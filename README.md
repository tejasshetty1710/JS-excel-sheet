# JS-excel-sheet
Basic Excel Sheet using Vanilla Js

## Features
- Add Column
- Add Row
- Add Column to the right and left
- Add Row to the right and left
- Delete a Column
- Delete a Row
- Add Column at the end
- Add Row at the end
- Auto Save
- Data Persistence using 'localStorage`
